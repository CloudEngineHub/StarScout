# Fake-Star-Detector

This is a project to detect fake stars in popular github repos

## Configuring Secrets in `secrets.yaml`

```
mongo_url: "your_mongo_url"
github_tokens:
  - token: "your_github_token"
    name: "your_github_username"
  - token: "your_github_token"
    name: "your_github_username"
```
